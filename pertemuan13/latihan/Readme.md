### Buatlah program PHP dengan ketentuan berikut:

1. Terdapat dua namespace berbeda:
   ```
   App\Produk
   App\Service
   ```
2. Di masing-masing namespace terdapat class dengan nama yang sama, yaitu **Item**
3. Setiap class Item memiliki:
   - Properti: nama
   - Method: info() yang menampilkan keterangan sesuai namespace

4. Buat instance dari kedua class Item tersebut dalam file utama, lalu tampilkan hasilnya
